IT-experience

Servers

I have been running my own machines for things like web servers, FTP servers, git repositories, etc. for a few years. As a result, I've become moderately confident working on remote machines via SSH, managing daemons and troubleshooting networks. My machines usually run Arch Linux, but I also have some experience with Debian-based systems.

Linux

I first heard about Linux in 2010 and immeadiately started playing around with the different distributions out there. I pretty quickly dual booted Ubuntu 10.10 on my computer - only keeping my current operating system for a few programs - and spent the next few months getting comfortable using the terminal and making the system work like I wanted before I, somewhat dissapointed about the lack of customizability - which was something I'd heard a lot about in regards to Linux - found Arch Linux, which I've been using as my only operating system for almost 4 years now.

Pair programming

Pair programming, for those uninitiated, is the act of programming together with another person on a single computer. I first came across it as an integral part of freeCodeCamp, and have since paired with people from all over the world while working through the FCC-curriculum.
